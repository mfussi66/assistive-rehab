Description

This module retrieves the transcript output from speech transcript and queries google cloud language services and provides the output on a port.

Parameters

  • –name: Module's name; all the open ports will be tagged with the prefix /name.
  • –language_code: Use English US
  • –sample_rate_hertz: Use 16000 hertz for the sample rate

Input Ports

  • /googleSpeechProcess/text:i [Bottle] [default carrier:]: Port to receive speech transcript to analyse

Output Ports

  • /googleSpeechProcess/result:o [Bottle] [default carrier:]: Sends the phrase analysis result of the output of google language query

Services

  • /googleSpeech/rpc [rpc-server]: service port . This service is described in googleSpeech_IDL (idl.thrift)
